__weakгҖҒ__blockе’Ң__strongдҝ®йҘ°з¬Ұ

еҸ‘иЎЁдәҺ 2020-02-24 жӣҙж–°дәҺ 2020-02-25 еҲҶзұ»дәҺ iOS
зӮ№еҮ»зңӢиҜҰжғ…

__weakжң¬иә«жҳҜеҸҜд»ҘйҒҝе…ҚеҫӘзҺҜеј•з”Ёзҡ„й—®йўҳзҡ„пјҢдҪҶжҳҜе…¶дјҡеҜјиҮҙеӨ–йғЁеҜ№иұЎйҮҠж”ҫдәҶд№ӢеҗҺпјҢblock еҶ…йғЁд№ҹи®ҝй—®дёҚеҲ°иҝҷдёӘеҜ№иұЎзҡ„й—®йўҳпјҢжҲ‘д»¬еҸҜд»ҘйҖҡиҝҮеңЁ block еҶ…йғЁеЈ°жҳҺдёҖдёӘ

__strongзҡ„еҸҳйҮҸжқҘжҢҮеҗ‘ ObjпјҢдҪҝеӨ–йғЁеҜ№иұЎж—ўиғҪеңЁ block еҶ…йғЁдҝқжҢҒдҪҸпјҢеҸҲиғҪйҒҝе…ҚеҫӘзҺҜеј•з”Ёзҡ„й—®йўҳгҖӮ

block жң¬иә«ж— жі•йҒҝе…ҚеҫӘзҺҜеј•з”Ёзҡ„й—®йўҳпјҢдҪҶжҳҜжҲ‘д»¬еҸҜд»ҘйҖҡиҝҮеңЁ block еҶ…йғЁжүӢеҠЁжҠҠ blockObj иөӢеҖјдёә nil зҡ„ж–№ејҸжқҘйҒҝе…ҚеҫӘзҺҜеј•з”Ёзҡ„й—®йўҳгҖӮеҸҰеӨ–дёҖзӮ№е°ұжҳҜblock дҝ®йҘ°зҡ„еҸҳйҮҸеңЁ block еҶ…еӨ–йғҪжҳҜе”ҜдёҖзҡ„пјҢиҰҒжіЁж„ҸиҝҷдёӘзү№жҖ§еҸҜиғҪеёҰжқҘзҡ„йҡҗжӮЈгҖӮ
жҖ»з»“пјҡ
1 . еңЁMRCпјҲжүӢеҠЁеҶ…еӯҳи®Ўж•°пјүжЁЎејҸдёӢпјҢ__blockдҝ®йҘ°пјҢеҸҜд»ҘйҒҝе…ҚеҫӘзҺҜеј•з”ЁпјӣARCпјҲиҮӘеҠЁеҶ…еӯҳи®Ўж•°пјүжЁЎејҸдёӢпјҢ__blockдҝ®йҘ°пјҢиҝҳжҳҜдјҡеј•иө·еҫӘзҺҜеј•з”Ёй—®йўҳпјӣ
2 . __blockдёҚз®ЎжҳҜARCиҝҳжҳҜMRCжЁЎејҸдёӢйғҪеҸҜд»ҘдҪҝз”ЁпјҢеҸҜд»Ҙдҝ®йҘ°еҜ№иұЎпјҢиҝҳеҸҜд»Ҙдҝ®йҘ°еҹәжң¬ж•°жҚ®зұ»еһӢпјӣ
3 . __weakеҸӘиғҪеңЁARCжЁЎејҸдёӢдҪҝз”ЁпјҢд№ҹеҸӘиғҪдҝ®йҘ°еҜ№иұЎпјҢдёҚиғҪдҝ®йҘ°еҹәжң¬ж•°жҚ®зұ»еһӢпјӣ
4 . __blockеҜ№иұЎеҸҜд»ҘеңЁblockдёӯиў«йҮҚж–°иөӢеҖјпјҢ__weakдёҚеҸҜд»Ҙпјӣ

API ReferenceеҜ№__blockеҸҳйҮҸдҝ®йҘ°з¬Ұзҡ„и§ЈйҮҠпјҢеӨ§жҰӮж„ҸжҖқпјҡ
1.__blockеҜ№иұЎеңЁblockдёӯжҳҜеҸҜд»Ҙиў«дҝ®ж”№гҖҒйҮҚж–°иөӢеҖјзҡ„гҖӮ
2.__blockеҜ№иұЎеңЁblockдёӯдёҚдјҡиў«blockејәеј•з”ЁдёҖж¬ЎпјҢд»ҺиҖҢдёҚдјҡеҮәзҺ°еҫӘзҺҜеј•з”Ёй—®йўҳгҖӮ

API ReferenceеҜ№__weakеҸҳйҮҸдҝ®йҘ°з¬Ұзҡ„и§ЈйҮҠпјҢеӨ§жҰӮж„ҸжҖқпјҡ
дҪҝз”ЁдәҶ__weakдҝ®йҘ°з¬Ұзҡ„еҜ№иұЎпјҢдҪңз”ЁзӯүеҗҢдәҺе®ҡд№үдёәweakзҡ„propertyгҖӮиҮӘз„¶дёҚдјҡеҜјиҮҙеҫӘзҺҜеј•з”Ёй—®йўҳпјҢеӣ дёәиӢ№жһңж–ҮжЎЈе·Із»ҸиҜҙзҡ„еҫҲжё…жҘҡпјҢеҪ“еҺҹеҜ№иұЎжІЎжңүд»»дҪ•ејәеј•з”Ёзҡ„ж—¶еҖҷпјҢејұеј•з”ЁжҢҮй’Ҳд№ҹдјҡиў«и®ҫзҪ®дёәnilгҖӮ
еӣ жӯӨпјҢ__blockе’Ң__weakдҝ®йҘ°з¬Ұзҡ„еҢәеҲ«е…¶е®һжҳҜжҢәжҳҺжҳҫзҡ„пјҡ
1.__blockдёҚз®ЎжҳҜARCиҝҳжҳҜMRCжЁЎејҸдёӢйғҪеҸҜд»ҘдҪҝз”ЁпјҢеҸҜд»Ҙдҝ®йҘ°еҜ№иұЎпјҢиҝҳеҸҜд»Ҙдҝ®йҘ°еҹәжң¬ж•°жҚ®зұ»еһӢгҖӮ
2.__weakеҸӘиғҪеңЁARCжЁЎејҸдёӢдҪҝз”ЁпјҢд№ҹеҸӘиғҪдҝ®йҘ°еҜ№иұЎпјҲNSStringпјүпјҢдёҚиғҪдҝ®йҘ°еҹәжң¬ж•°жҚ®зұ»еһӢпјҲintпјүгҖӮ
3.__blockеҜ№иұЎеҸҜд»ҘеңЁblockдёӯиў«йҮҚж–°иөӢеҖјпјҢ__weakдёҚеҸҜд»ҘгҖӮ

Tipsпјҡ__unsafe_unretainedдҝ®йҘ°з¬ҰеҸҜд»Ҙиў«и§ҶдёәiOS SDK 4.3д»ҘеүҚзүҲжң¬зҡ„__weakзҡ„жӣҝд»Је“ҒпјҢдёҚиҝҮдёҚдјҡиў«иҮӘеҠЁзҪ®з©әдёәnilгҖӮжүҖд»Ҙе°ҪеҸҜиғҪдёҚиҰҒдҪҝз”ЁиҝҷдёӘдҝ®йҘ°з¬ҰгҖӮ
Blockзҡ„з”ҹе‘Ҫе‘Ёжңҹз®ЎзҗҶйқһеёёзҡ„еҫ®еҰҷпјҢдёҺARCж··еңЁдёҖиө·еҗҺпјҢжӣҙеҠ еӨҚжқӮгҖӮ
гҖҖгҖҖеҪ“Blockе»¶stackеҗ‘дёҠпјҲupпјүдј йҖ’зҡ„ж—¶еҖҷпјҢзӣҙжҺҘиҝ”еӣһпјҢзј–иҜ‘еҷЁдјҡж·»еҠ [[ copy] autorelease]д»Јз ҒгҖӮ
гҖҖгҖҖеҪ“Blockе»¶stackеҗ‘дёӢдј йҖ’з»ҷйңҖиҰҒretainзҡ„е®№еҷЁзҡ„ж—¶еҖҷпјҢйңҖиҰҒжҳҫејҸзҡ„и°ғз”Ё[^{} copy]ж–№жі•гҖӮ
гҖҖ еңЁARCдёӢпјҢ__blockдҝ®ж”№зҡ„NSObjectжҢҮй’Ҳдҫқз„¶дјҡиў«retainгҖӮ
еңЁARCдёӢпјҢдёҖдёӘblockеҶ…еј•з”ЁдёҖдёӘеҜ№иұЎзҡ„е®һдҫӢеҸҳйҮҸеҗҺпјҢselfдјҡиў«retainпјҢжүҖд»ҘжһҒжҳ“йҖ жҲҗstrong reference cycleпјҢеҸҜд»ҘйҖҡиҝҮ__weakжҢҮй’ҲжқҘйҒҝе…Қиҝҷз§Қжғ…еҪўпјҢеӣ дёәARCдёҚдјҡдёә__weakжҢҮй’ҲretainгҖӮ
